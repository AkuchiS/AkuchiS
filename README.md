# Hi, I'm AkuchiS 👋

I build practical tools for people who run their own AI, automation, and
self-hosted setups — then I open-source the ones that earn their place.

Everything here was built on my own infrastructure: broken, fixed, and hardened in
daily use before it ever went public. I'm a retired Australian Army veteran,
building independently and self-funded — no VC, no growth team, just tools that work.

**Receipts over promises.** Most of these ship a `selftest` or a one-command
verifier, so you don't have to take my word for anything — you run it and watch the
numbers on your own machine.
