## About AkuchiS

I’m a retired Australian Army veteran, homelab hobbyist, and independent builder exploring practical uses for AI, automation, and self-hosted technology.

I build tools that solve problems I actually have, test them on my own infrastructure, break plenty of things along the way, fix what fails, and share the results so others can benefit.

**Simplify. Automate. Share.**

### What's here

| Project | What it does | |
|---|---|---|
| [yap](https://github.com/AkuchiS/yap) | Offline voice dictation for macOS/Windows/Linux — hold a key, speak, your words land at the cursor. Local Whisper, learns your vocabulary. | Free · MIT |
| [FlowProof](https://github.com/AkuchiS/flowproof) | Checks an n8n workflow export imports cleanly — leaked creds, missing nodes, version drift, broken connections. | Free · MIT |
| [n8n-nodes-flowproof](https://github.com/AkuchiS/n8n-nodes-flowproof) | FlowProof as an n8n community node — validate workflows from inside n8n. | Free · MIT |
| [AEGIS Guard](https://github.com/AkuchiS/aegis-guard) | Drop-in input/output safety for AI agents — screens prompt-injection in, catches leaked secrets and refusals out. Zero dependencies. | Free · MIT |
| [Governance-in-a-Box](https://github.com/AkuchiS/governance-in-a-box) | Verifiable AI-build governance — modules that prove themselves with one offline command. | Free · MIT |
| [ProxSweep](https://github.com/AkuchiS/proxsweep) | One script updates your whole Proxmox homelab — host, every LXC, and Docker — in a single pass. | Free · MIT |
| [pinchflat](https://github.com/AkuchiS/pinchflat) | Installs Pinchflat (self-hosted YouTube archiver) in a Proxmox LXC with one command. | Free · MIT |
| [n8n-templates](https://github.com/AkuchiS/n8n-templates) | Production-shaped n8n workflows, each validated import-clean. | Free |

Everything runs on your own machine — local-first, no telemetry. Most of it is independently developed, self-funded, and released openly under MIT.
